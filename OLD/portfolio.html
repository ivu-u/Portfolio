Bonbon

My Roles: Programming Co-lead, Gameplay Programmer


Bonbon is a 3D story-driven RPG about a boy who wants to follow his dreams, and his older sister who just wants to keep him safe. The main gameplay loop consists of turn-based combat interspersed with cute dialogue scenes.


Bonbon is the project I’m currently working on through the Georgia Tech Game Development Club (VGDev)! Learn more about the project and my roles below: (link)

IKA

My Roles: Game/Style Designer, Gameplay Programmer


IKA is a 3D boss battle demonstration using Inverse Kinematics to control the octopus’s tentacles. The art style is based off Takashi Murakami’s works, distorting cute images into something disturbing.


I’m continuing work learning Inverse Kinematics in Unity, and plan to add them to Bonbon to animate one of our enemies, the crystal centipede.


The game was made for the 2023 AtlantaDev game jam.

VotingSim

My Roles: Pixel-Artist, Gameplay Designer, Level Creator


VotingSim is a top-down RPG that uses WarioWare style gameplay to teach players all the steps to register to vote! This game was made for the Games For Change 2022 Challenge, and was a finalist for the state of Georgia. The theme for this challenge was “Voices of a New Generation…”


I used Aseprite and Unity to create and integrate every 16-bit character sprite and animation, in addition to crafting the top-down tile sets employed in level design.

Wave Function Collapse Algorithm

My Roles: Algorithm Architecture Co-Designer, Programmer


Wave Function Collapse is an algorithm used to generate procedural levels in games. The small game “Cyber Chase” is a prototype of this algorithm that a friend and I developed to practice designing complex algorithms. It’s a simple game where the focus is to find a key that was generated in a random part of the map and reach the exit without being killed by enemies.
